## 学习页（课程Tab）接口文档

---

## 1. 获取学习页筛选项（右侧筛选面板）

### 请求信息

- **接口地址**：`/api/app/v1/study/course/filters`
- **请求方式**：GET
- **接口说明**：获取学习页右侧「筛选」面板的数据源。前端会自动在首位补一个 `全部` 选项。

### 请求参数

无

### 响应数据

```json
{
  "code": 200,
  "message": "success",
  "data": [
    { "label": "唱歌", "value": "sing" },
    { "label": "特训营", "value": "camp" },
    { "label": "气血调理", "value": "qi" },
    { "label": "手机摄影", "value": "photo" }
  ]
}
```

字段说明：

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| label | string | 展示文案 |
| value | string | 筛选值（唯一） |
| tip | string | 可选提示文案 |

---

字段说明：

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| label | string | 展示文案 |
| value | string | 类型筛选值（唯一） |
| tip | string | 可选提示文案 |

## 4. 获取学习页默认分组数据（总览态）

### 请求信息

- **接口地址**：`/api/app/v1/study/course/sections`
- **请求方式**：GET
- **接口说明**：用于学习页默认状态下展示「最近学习 / 待学习 / 已结课」三组数据。

### 请求参数

无

### 响应数据

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "recentSection": {
      "title": "最近学习",
      "list": [
        {
          "id": 11,
          "title": "1月7日：小寒养生3步走",
          "cover": "https://example.com/cover.jpg",
          "overlayText": "01月07日 19:00",
          "actionText": "继续学"
        }
      ]
    },
    "pendingSection": {
      "title": "待学习",
      "list": []
    },
    "finishedSection": {
      "title": "已结课",
      "list": []
    }
  }
}
```

说明：

- `recentSection / pendingSection / finishedSection` 允许按需返回；不返回或 `list` 为空时前端不展示该分组。
- 为兼容旧数据结构，也支持返回 `recentList / pendingList / finishedList`。

## 5. 获取学习页筛选后的课程列表（分页）

### 请求信息

- **接口地址**：`/api/app/v1/study/course/list`
- **请求方式**：GET
- **接口说明**：学习页筛选后列表查询，支持分页。

### 请求参数（Query）

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| filter | string | 否 | 右侧「筛选」面板的选中值（如 `sing` / `camp`）；未选择时传空字符串 `""` |
| filterType | string | 否 | 左侧「全部类型」面板的选中值（如 `open` / `trial` / `formal`）；未选择时传空字符串 `""` |
| page | number | 是 | 当前页码，从 1 开始 |
| pageSize | number | 是 | 每页条数 |

参数组合说明：

- 该接口支持左右筛选组合查询。
- 示例1：筛选“体验营下唱歌”可传 `filter=sing&filterType=trial`。
- 示例2：只筛“唱歌”可传 `filter=sing&filterType=`。
- 示例3：只筛“体验营”可传 `filter=&filterType=trial`。

### 响应数据

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 101,
        "title": "才艺名师公开课",
        "cover": "https://example.com/cover.jpg",
        "overlayText": "公开课",
        "statusText": "2026.03.01 19:00 开课",
        "actionText": "去学习"
      }
    ],
    "total": 38,
    "page": 1,
    "pageSize": 10
  }
}
```

字段说明：

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| id | number | 课程ID |
| title | string | 课程标题 |
| cover | string | 封面图地址 |
| overlayText | string | 封面底部遮罩文案 |
| timeText | string | 可选时间文案 |
| statusText | string | 状态文案 |
| actionText | string | 可选按钮文案 |

---

## 通用响应约定

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

- 成功码兼容 `200` 或 `0`。
- 失败时返回统一错误码（如 `400`、`401`、`500`）和可读 `message`。
