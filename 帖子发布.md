## 帖子发布模块接口文档（v1）

### 发布文章

- **接口**：`POST /api/app/v1/post/store/article`
- **说明**：发布文章（`postType=3`），支持正文中混排文字/图片/视频。

#### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| title | string | 是 | 文章标题 |
| content | string | 是 | 文章正文（建议 HTML 富文本） |
| images | string[] | 否 | 正文中提取的图片 URL 列表 |
| videoUrls | string[] | 否 | 正文中提取的视频 URL 列表 |
| topics | Topic[] | 否 | 话题列表 |

#### 请求示例

```json
{
  "title": "春季养生食谱分享",
  "content": "<p>今天分享三道简单养生餐...</p><img src=\"https://example.com/a.jpg\" /><video src=\"https://example.com/v.mp4\"></video>",
  "images": [
    "https://example.com/a.jpg"
  ],
  "videoUrls": [
    "https://example.com/v.mp4"
  ],
  "topics": [
    {
      "id": 5,
      "name": "#你好2028"
    }
  ]
}
```

#### 响应示例

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "postId": 350785364799
  }
}
```
