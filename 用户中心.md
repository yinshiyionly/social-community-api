# 用户模块接口文档

---

## 1. 获取个人资料

### 请求信息

- **接口地址**：`/api/app/v1/user/profile`
- **请求方式**：GET
- **接口说明**：获取指定用户的详细资料，包括头像、背景图、各项统计数据及学分。

#### 请求参数（Query）

| 参数名     | 类型   | 必填 | 说明                |
| ---------- | ------ | ---- | ------------------- |
| `memberId` | string | 是   | 用户唯一标识 ID     |

### 响应数据

#### 成功响应

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "nickname": "墨雨云",
    "avatar": "/static/images/avatar.jpg",
    "bgImage": "/static/images/homepage-bgImage.png",
    "followCount": 1,
    "fansCount": 1,
    "likeCount": 30,
    "score": 106
  }
}
```

#### 字段说明

| 字段名        | 类型   | 说明                                     |
| ------------- | ------ | ---------------------------------------- |
| `nickname`    | string | 用户昵称                                 |
| `avatar`      | string | 用户头像 URL                             |
| `bgImage`     | string | 个人主页背景图 URL                       |
| `followCount` | number | 关注数                                   |
| `fansCount`   | number | 粉丝数                                   |
| `likeCount`   | number | 获赞总数                                 |
| `score`       | number | 当前学分                                 |

## 2. 获取个人帖子列表（分页）

### 请求信息

- **接口地址**：`/api/app/v1/user/posts`
- **请求方式**：GET
- **接口说明**：分页获取指定用户发布的帖子列表，用于个人主页「帖子」Tab 展示。

#### 请求参数（Query）

| 参数名      | 类型   | 必填 | 说明                |
| ----------- | ------ | ---- | ------------------- |
| `memberId`  | string | 是   | 用户唯一标识 ID     |
| `page`      | number | 是   | 当前页码，从 1 开始 |
| `pageSize`  | number | 是   | 每页条数            |

---

## 3. 获取个人收藏列表（分页）

### 请求信息

- **接口地址**：`/api/app/v1/user/favorites`
- **请求方式**：GET
- **接口说明**：分页获取指定用户收藏的帖子列表，用于个人主页「收藏」Tab 展示。

#### 请求参数（Query）

| 参数名      | 类型   | 必填 | 说明                |
| ----------- | ------ | ---- | ------------------- |
| `memberId`  | string | 是   | 用户唯一标识 ID     |
| `page`      | number | 是   | 当前页码，从 1 开始 |
| `pageSize`  | number | 是   | 每页条数            |

### 响应数据

#### 成功响应

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "id": "5001",
        "title": "【收藏】非常有用的教程",
        "cover": "https://example.com/fav-cover.jpg",
        "aspectRatio": 1.2,
        "isVideo": true,
        "avatar": "/static/images/author.jpg",
        "nickname": "知识博主",
        "likes": 560,
        "views": 3200
      }
    ],
    "total": 15,
    "page": 1,
    "pageSize": 10
  }
}
```

---

## 数据结构说明

### 瀑布流卡片对象 (FeedItem)

| 字段名        | 类型    | 说明                                     |
| ------------- | ------- | ---------------------------------------- |
| `id`          | string  | 帖子唯一标识                             |
| `title`       | string  | 帖子标题                                 |
| `cover`       | string  | 封面图 URL                               |
| `aspectRatio` | number  | 封面图宽高比（宽/高），用于瀑布流高度计算 |
| `isVideo`     | boolean | 是否为视频类型                           |
| `avatar`      | string  | 作者头像 URL                             |
| `nickname`    | string  | 作者昵称                                 |
| `likes`       | number  | 点赞数                                   |
| `views`       | number  | 浏览量                                   |
