## 1. 获取入学信息详情

### 请求信息

- **接口地址**：`/api/app/v1/course/enrollment-detail`
- **请求方式**：GET
- **接口说明**：用于课程「入学信息」页面，返回用户手机号和支付金额。

### 请求参数（Query）

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | number | 是 | 课程 ID |

### 请求示例

```http
GET /api/app/v1/course/enrollment-detail?id=1001
```

### 响应示例

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "phone": "15122443037",
    "payAmount": "1.00"
  }
}
```

### 响应字段说明

顶层字段：

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| code | number | 状态码，`200` 表示成功 |
| message | string | 响应消息 |
| data | object | 业务数据 |

`data` 字段：

| 字段名 | 类型 | 必返 | 说明 |
| --- | --- | --- | --- |
| phone | string | 否 | 手机号（用于入学信息展示） |
| payAmount | string | 否 | 支付金额；为空字符串 `""` 或不返回表示免费领取 |

### 业务约定

- `payAmount` 有值（如 `"1.00"`）表示需支付对应金额。
- `payAmount` 为空字符串或缺省，前端按“免费领取”处理。

### 错误响应示例

```json
{
  "code": 400,
  "message": "参数错误",
  "data": null
}
```
