# 课程页面接口文档

## 1. 获取课程分类列表

### 请求信息
- **接口地址**：`/api/course/categories`
- **请求方式**：GET
- **请求参数**（Query，均为可选）：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| limit | number | 否 | 限制返回条数。**传该参数**时只返回前 `limit` 条（如选课页传 `limit=4` 只返回 4 条）；**不传**时返回全部分类（如选课中心左侧列表） |

### 响应数据

#### 成功响应
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 1,
      "name": "睡眠调理",
      "icon": "/static/svg/icon-sleep.svg"
    },
    {
      "id": 2,
      "name": "传统健身",
      "icon": "/static/svg/icon-Tradition.svg"
    }
  ]
}
```

#### 字段说明（data 数组元素）
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | number | 分类ID，唯一标识 |
| name | string | 分类名称 |
| icon | string | 分类图标（图片 URL 或前端图标名） |

---

## 2. 按分类获取课程列表（分组）

### 请求信息
- **接口地址**：`/api/course/list`
- **请求方式**：GET
- **请求参数**（Query）：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| categoryId | number | 是 | 分类ID，对应「课程分类列表」中的 `id` |

### 响应数据

#### 成功响应
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "typeName": "招生0元课",
      "typeIcon": "/static/svg/icon-free.svg",
      "typeIconColor": "#4A90E2",
      "list": [
        {
          "id": 101,
          "cover": "https://example.com/course1.jpg",
          "title": "小寒养生3步走 练好吃好睡好...",
          "desc": "副标题描述不超10个字",
          "price": "0",
          "originalPrice": "99"
        }
      ]
    },
    {
      "typeName": "进阶课",
      "typeIcon": "/static/svg/icon-advance.svg",
      "typeIconColor": "#E62E2E",
      "list": [
        {
          "id": 102,
          "cover": "https://example.com/course2.jpg",

          "desc": "适合有基础学员",
          "price": "1.00",
          "originalPrice": "199"
        }

    }

```

#### 字段说明

**data 数组元素（分组对象）：**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| typeName | string | 是 | 分组名称，如「招生0元课」「进阶课」「高阶课」等，由后端决定 |
| typeIcon | string | 否 | 分组标题前图标（图片 URL） |
| typeIconColor | string | 否 | 标题/图标强调色，如 `#4A90E2`、`#E62E2E` |
| list | array | 是 | 该分组下的课程列表，见下表 |

**list 数组元素（课程对象）：**

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | number | 课程ID |
| cover | string | 课程封面图片URL |
| title | string | 课程标题 |
| desc | string | 课程描述/副标题 |
| price | string | 现价（元） |
| originalPrice | string | 原价（元） |

## 3. 获取课程详情（课程详情页使用）

### 请求信息
- **接口地址**：`/api/course/detail`
- **请求方式**：GET
- **请求参数**（Query）：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | number | 是 | 课程ID |

### 响应数据

- **内容区**：由前端用返回的 **contentImage** 一张图展示，不做富文本。
- **底部栏**：固定底栏；限时价、原价、优惠积分 **有则显示、无则不显示**；右侧按钮文案与点击行为由后端返回字段决定。

#### 成功响应
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "contentImage": "https://example.com/course-detail-banner.jpg",
    "limitPrice": "0",
    "originalPrice": "999.00",
    "discountPoints": "100",
    "buttonText": "免费领取课程",
    "buttonActionType": "free_receive"
  }
}
```

#### 字段说明（data 对象）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| contentImage | string | 是 | 详情内容区图片 URL（整张图，前端仅展示此图） |
| limitPrice | string | 否 | 限时价格，有则显示在底部左侧 |
| originalPrice | string | 否 | 原价，有则显示（可划线） |
| discountPoints | string | 否 | 优惠积分文案，有则显示（如 "100积分"） |
| buttonText | string | 是 | 底部右侧按钮文案，如「免费领取课程」「立即购买」 |
| buttonActionType | string | 否 | 按钮点击行为类型，前端据此分支：如 `free_receive` 免费领取、`buy` 立即购买等 |

## 4. 获取大咖直播列表

### 请求信息
- **接口地址**：`/api/course/live-list`
- **请求方式**：GET
- **请求参数**：无

### 响应数据

```json
{

  "message": "success",
  "data": [
    {
      "id": 1,
      "cover": "https://example.com/cover.jpg",
      "liveTime": "1月5日 19:00",
      "title": "东莞实践课堂，三维码科技课堂",
      "viewCount": 161,
      "replayUrl": "https://example.com/replay/123"
    },
    {
      "id": 2,
      "cover": "https://example.com/cover2.jpg",
      "liveTime": "1月6日 20:00",
      "title": "智慧课堂教学实践分享",
      "viewCount": 203,
      "replayUrl": ""
    }
  ]

```

#### 字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | number | 直播ID |
| cover | string | 封面图片URL |

## 通用说明

### 错误响应
```json
{
  "code": 500,
  "message": "服务器错误",
  "data": null
}
```

### 常见错误码
| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

### 注意事项
1. 所有接口返回的图片URL需要是完整的HTTPS地址
2. 价格字段为字符串类型，保留两位小数
3. 直播时间格式统一为"月日 时:分"（如：1月5日 19:00）
4. replayUrl 为空字符串表示无回放，前端不显示"看回放"按钮
